# Security Policy

## Melden von Sicherheitsproblemen

Bitte Sicherheitsprobleme nicht oeffentlich als Issue diskutieren. Intern pruefen und beheben, bevor Details veroeffentlicht werden.

## Datenschutzrelevante Grundsaetze

- Die Fotoanalyse findet lokal im Browser statt.
- Es werden keine Kleidungsfotos an einen Server gesendet.
- Es gibt keine API-Schluessel im Frontend.
- Externe Links, z. B. Telegram, werden klar als externe Navigation behandelt.

## Abhaengigkeiten

Dependabot ist fuer npm, Python-Dev-Abhaengigkeiten und GitHub Actions aktiviert. Sicherheitsupdates sollten bevorzugt behandelt werden.
